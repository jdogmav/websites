Final Project

Minimum Requirements

  • Readability and Intelligibility of code(class & id naming conventions, properly commented code, nesting code, etc.)
  • Directory structure
  • Everything is linked correctly (meta tags, link tags, order of stylesheets)
  • All Anchor tags function and make sense
  • Have your code added to a repository on Github.com

Content

  • Build a Website with:
    • 3 Pages: Home, Contact, About
    • Header with Brand and Working Navigation
    • Footer with Social Media Links from Font Awesome that function properly
    • Working Contact Form from Wufoo
    • At least one image that is a link
    • At least one set of columns
    • Locally downloaded dependencies
    • Incorporate Google Analytics
    • Get Creative!

Design

  • Wireframe
  • Working Navigation
  • Hero Image & call to action
  • Use hover pseudo selector on two different types of HTML elements
  • Color Scheme and consistent design across all pages that makes sense/is complimentary
  • At least one Google Map Embed
  • Either a video or an audio html tag that works
